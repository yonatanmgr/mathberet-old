כדי להתחיל:

לחצו על כפתור המחברות שבצד ימין ( O+ Ctrl ) ובחרו קובץ או מחברת,
או לחלופין לחצו על כפתור "צור קובץ חדש" שבצד ימין ( N+ Ctrl )

שכחתם קיצור מקלדת? תמיד תוכלו ללחוץ על כפתור העזרה ( H+ Ctrl ) ולהיזכר!
